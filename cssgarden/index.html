Title

This is a personal CSS Garden, inspired by the CSS Zen Garden. The goal of these pages is to create a series of different layouts and themes, all using the same base HTML. If one page has it, they all have it. The personally imposed limitation is to only change the CSS file and the title of the page. Unless otherwise stated on the garden's index, the only boilerplate CSS I'm using is Skeleton, with some basic classes I created for ease of formatting.
