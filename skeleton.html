This page was developed using the Skeleton framework. It's simple, but that's the point!

Skeleton is a lightweight CSS boilerplate for responsive web design. It comes with a responsive grid and some light styling, but most features are up to the developer to create or add. By doing this we can prevent bringing in a load of JavaScript and CSS for a simple page-- like this one-- and we can still enjoy a mobile responsive framework.

I've created a small CSS file to add back some features from Bootstrap that are not found in Skeleton by default. I've also added in a navigation bar above.


The images above are simple circle and rounded classes, using the u-max-full-width class from Skeleton to make them display responsively. The image to the right is using a pull-right class. If you're viewing this on a mobile device or resize your browser's window they should all shrink down and continue to display properly, while being pulled onto their own lines for better viewing.

With the snippets of code I've assembled from this test-run I should be able to develop lighter pages that are both cleaner to read from a developer's standpoint, and more efficient for users to access.
