Brewing up a Storm!

Welcome

This design was created in WordPress, using the XTheme’s Icon stack. It has been rendered here as a static page with the help of the Simply Static static page generator. I have edited the resulting output only lightly, mainly for the sake of readability. The stock logo on the header of this page is from Free-Design-Logo.net. Even though I’m not a coffee drinker I thought it looked great to base a design around! The majority of the design for this page was based around the logo through it’s colors and typeface, using shades of brown to complement it. This design is aimed at a smaller restaurant, one that likely only needs information displayed (thus the light navbar above).
